# Portfolio Website

![Site preview](/preview.png)

A personal portfolio to showcase a few projects.

## Install & run

```bash
yarn && yarn start
```

## Setup Firebase

```json

// In src/config.json

{
  "firebase": {
    "apiKey": "project-key",
    "databaseURL": "https://domain.firebaseio.com",
    "projectId": "project-id",
  }
}
```
